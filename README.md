# 🎁 Raksha Bandhan Greetings Website

A small fun web project created for **Raksha Bandhan 2025** to send personalized greeting pages to friends.

The website maps specific **usernames to custom greeting pages**, allowing each person to receive their own festive message. The project was briefly hosted using **GitHub Pages** so friends could access their greeting links during the festival.

---

## ✨ Features

- 🎀 Personalized Raksha Bandhan greeting pages  
- 🔗 Username → custom HTML page mapping  
- 🌐 Static website hosted using **GitHub Pages**  
- ⚡ Lightweight and easy to extend  
- 🎉 Simple and fun way to share festive messages
